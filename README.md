# Projet Test Framework

## Description
Ce projet est un framework de test conçu pour démontrer la gestion des entités `Region` et `Territorie` via une architecture de contrôleur REST. Le projet est construit en utilisant le framework Spring Boot et il englobe des fonctionnalités pour gérer les entités persistantes.

## Structure du Projet
- **Controllers**: Gèrent les requêtes HTTP et effectuent des opérations CRUD sur les entités.
- **Entities**: Représentent les données du domaine travail avec la base de données.

## Prérequis
- Java 11 ou supérieure
- Maven 3.6 ou supérieure
