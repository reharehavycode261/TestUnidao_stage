# Projet Example

Ce projet est une application Java conçue pour démontrer une architecture basée sur le modèle MVC avec une utilisation des annotations pour simplifier le développement de REST APIs. Le projet met en œuvre des interactions de base avec une base de données à l'aide de la bibliothèque `uniDao`.

## Structure du projet

- **src/** : Contient le code source de l'application.
  - **Main.java** : Point d'entrée principal de l'application.
  - **project/** : Contient les classes de contrôleurs et de modèles.
- **test-framework/** : Contient les tests unitaires et fichiers associés utilisés pour vérifier les fonctionnalités.

## Prérequis

- Java 11 ou supérieur
- Maven 3.6.x ou supérieur
- Accès à une base de données compatible SQL
