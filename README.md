# Project Overview

Welcome to the Project! This project is built with Java and Spring Boot, following the MVC architecture. Below is an overview of the features currently implemented in the project.

## Features

1. **Advanced Search Functionality**
   - Includes filters, auto-completion, and query saving.
   - Dependency: Spring Data JPA.

2. **Multi-Channel Notifications**
   - In-app, email, and push notifications with user preferences.
   - Dependencies include Spring Mail and Firebase Cloud Messaging.

3. **Analytical Dashboard and Data Export**
   - Provides analytics and supports exporting data in CSV/Excel format.

4. **Internationalization and Dark Mode**
   - Supports multiple languages (i18n) and offers a user-switchable dark mode theme.

5. **Feature Flags Management**
   - Implements feature flag management with progressive deployments.

6. **Performance and Error Monitoring**
   - Metrics, logs, traces with alerting capabilities have been integrated.

7. **End-to-End Testing & CI/CD Pipeline**
   - E2E tests and a CI/CD pipeline are set up, including automated security scans.

8. **Client-side Sorting and Pagination**
   - Enabled sorting by columns with client-side pagination on lists.

9. **UI/UX Enhancements**
   - Unified empty states, loaders, and error messages using simple components.

## Technology Stack

- **Backend:** Java, Spring Boot
- **Database:** Integrated with Spring Data JPA for database operations.
- **Frontend:** Utilizes standard web technologies for UI components.
- **Messaging:** Firebase Cloud Messaging for push notifications.
- **Email:** Spring Mail for handling email notifications.
