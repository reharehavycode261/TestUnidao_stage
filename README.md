# Projet MVC en Java et JavaScript

Ce projet utilise une architecture MVC (Model-View-Controller) avec Java pour le back-end et JavaScript pour le front-end. Il est construit à l'aide de Spring Boot pour garantir une intégration transparente et un développement rapide.

## Fonctionnalités principales

1. **Architecture MVC**: Utilisation de contrôleurs, de modèles et de vues pour séparer les responsabilités et améliorer la maintenabilité du code.

2. **Recherche avancée**: 
   - Intégration de fonctionnalités de recherche avancée avec filtres, auto-complétion et sauvegarde des requêtes.
   - Permet aux utilisateurs de rechercher des territoires et des régions de manière plus efficace.

3. **Notifications multi-canaux**:
   - Système de notification via différents canaux comme in‑app, e‑mail, et notifications push.
   - Les utilisateurs peuvent personnaliser leurs préférences de notification.

4. **Analytical Dashboard**:
   - Offre un tableau de bord analytique et l'exportation de données en CSV/Excel pour un suivi détaillé des performances.

5. **Internationalisation et mode sombre**:
   - Support des langues multiples (i18n) pour une expérience utilisateur améliorée.
   - Option mode sombre pour réduire la fatigue oculaire lors de l'utilisation de l'application.

6. **Gestion des fonctionnalités**:
   - Implémentation de feature flags pour permettre des déploiements progressifs sans interruption.

7. **Monitoring de performance**:
   - Intégration de métriques, logs et traces avec alertes pour superviser les performances et anticiper les pannes.

8. **Tests et CI/CD**:
   - Tests E2E et pipeline CI/CD avec scans de sécurité automatisés pour garantir un déploiement robuste et sécurisé.

9. **Tri et pagination côté client**:
   - Activation du tri par colonnes et pagination pour gérer les longues listes de données.

10. **Composants unifiés pour états d'interface**:
    - États vides, loaders et messages d’erreur sont uniformisés pour une expérience utilisateur cohérente.

## Technologies utilisées

- **Back-end**: Java, Spring Boot
- **Front-end**: JavaScript
- **Base de données**: PostgreSQL
