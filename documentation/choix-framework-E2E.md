# Choix du Framework de Tests E2E

Pour les tests End-to-End, nous avons choisi d'utiliser **Cypress**. Voici pourquoi :

- **Facilité d'utilisation** : Cypress est facile à installer et à utiliser grâce à sa documentation exhaustive et un large soutien communautaire.
- **Intégration** : S'intègre bien avec de nombreux outils CI/CD et plateformes cloud.
- **Rapidité** : Exécute les tests rapidement, ce qui est idéal pour les itérations rapides.
- **Debugging** : Offre d'excellents outils de debugging, ce qui réduit le temps de correction des erreurs.

Cette décision permettra d'avoir une base solide pour écrire des tests robustes et facilement maintenables.
